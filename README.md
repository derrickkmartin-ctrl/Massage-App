# Session Notes — Setup Guide

A mobile-first PWA for bodywork practitioners. Post-session notes → SOAP note + client home care email, generated by AI.

---

## What's in this folder

- `index.html` — the entire app (one file)
- `manifest.json` — makes it installable as a home screen app
- `icon.svg` — app icon
- `README.md` — this file

---

## 3-step setup

### Step 1 — Get your Anthropic API key (2 min)

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Sign up or log in
3. Click **API Keys** in the left sidebar
4. Click **Create Key**, give it a name (e.g. "Session Notes App")
5. Copy the key — it starts with `sk-ant-`
6. **Recommended:** Go to Billing → set a monthly spend limit (e.g. $5)

---

### Step 2 — Host on GitHub Pages (5 min)

1. Go to [github.com](https://github.com) and create a free account if you don't have one
2. Click the **+** icon (top right) → **New repository**
3. Name it `session-notes` (or anything you like)
4. Set it to **Public**
5. Click **Create repository**
6. Click **uploading an existing file**
7. Drag all 4 files from this folder into the upload area
8. Click **Commit changes**
9. Go to **Settings** → **Pages** (left sidebar)
10. Under "Branch," select `main` → click **Save**
11. Wait ~60 seconds, then your URL will appear:
    `https://YOUR-USERNAME.github.io/session-notes/`

That's your app URL. Bookmark it.

---

### Step 3 — Add to your home screen

**On iPhone (Safari):**
1. Open your app URL in Safari
2. Tap the Share button (box with arrow pointing up)
3. Scroll down → tap **Add to Home Screen**
4. Tap **Add**

**On Android (Chrome):**
1. Open your app URL in Chrome
2. Tap the three-dot menu
3. Tap **Add to Home screen**

---

### Step 4 — Enter your API key in the app

1. Open the app from your home screen
2. Tap the ⚙ icon (top right)
3. Paste your API key
4. Add your name (for email sign-offs)
5. Tap **Save settings**

Done. Your key is stored only on your device.
